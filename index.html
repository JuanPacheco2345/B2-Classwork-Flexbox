What is Flex Box?

A CSS layout mode that provides an easy and clean way to arrange items within a container
  • NO FLOATS!
  • Responsive and Mobile Friendly
  • Position of child elements is much EASIER
  • Flex container margins do not collapse with the margins of its contents
  • Order of elements can easily be changed without editing the HTML file


FLEX BOX MODEL CONCEPT

  • The ability to alter item width and height to best fit in its containers avaiable free space
  • Flexbox is direction-agnostic
  • Position of child elements is much EASIER
  • Built for small-scale layouts

Justify-Content

  • This defines the alignment along the main axis. It helps distribute extra free space left over when either all the flex items on a line are inflexible, or are flexible but have reached their maximum size. It also exerts some control over the alignment of items when they overflow the line.

Align-Items

  • This defines the default behavior for how flex items are laid out along the cross axis on the current line. Think of it as the justify-content version for the cross-axis (perpendicular to the main-axis).
